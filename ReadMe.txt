Brackets is a simple program completed for a "hard" programming challenge on Dcoder called "The Game of Brackets"

Input: Contains a string

Output: Print "Yes" for a valid string, else "No"
Examples: 
{{[]}}: Yes
{[}]: No
{[]): No
{}([]): Yes

Constraints: 
0<=string length>=100
Parenthesis for this problem includes "[({})]"

Sample Input:
[[{{()}}]]

Sample Output:
Yes
